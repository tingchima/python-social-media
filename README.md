# `python-social-media`

Implement simple social media application with Python and Django

## Feature

1. user create
2. chatroom create
3. chatroom get
4. chatrooms of user
5. chatroom join
6. message create
7. messages of room
8. message delete

## Dependencies

1. Django
2. Django-ninja
3. channels
4. PostgreSQL
5. DynamoDB
6. PyJWT
7. RabbitMQ
8. Pika
